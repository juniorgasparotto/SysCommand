# Instalação <header-set anchor-name="install" />

Via [NuGet](https://www.nuget.org/packages/SysCommand/):

```
Install-Package SysCommand
```

_Nota: O pacote traz o arquivo `Program.cs.txt` que contém um template básico. Para utiliza-lo, substitua o conteúdo do arquivo `Program.cs` pelo conteúdo deste arquivo._
